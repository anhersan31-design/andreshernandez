3 Must Have
Feature 1:  Dynamic Weather Backgrounds
Feature 2: Multi-City Sports Venue Tracker
Feature 3: "Fit-Weather" Activity Suggester

Description: The **Guapurri Weather App** is designed to transform raw JSON data into a high-end, user-centric experience. 

1. **Dynamic Backgrounds:** The UI will automatically change its theme based on real-time conditions (e.g., sunny, rainy, or cloudy) using the OpenWeatherMap API data.
2. **Sports Venue Tracker:** Users can monitor weather specifically for major sports stadiums globally—perfect for checking if a match in Barcelona or an NFL game will be affected by rain.
3. **Fit-Weather Suggester:** A smart engine that analyzes temperature and humidity to recommend the best time for a run or the gym, even suggesting a specific healthy smoothie recipe based on the outdoor heat.
